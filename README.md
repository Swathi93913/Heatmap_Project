# Correlation Heatmap of Student Scores

This project demonstrates how to create a **correlation heatmap** using Python to understand relationships between different variables in a dataset.

## 📊 Project Overview
The heatmap visualizes correlations between:
- Math Score
- Reading Score
- Writing Score

## 🛠 Tools & Libraries Used
- Python
- Pandas
- Seaborn
- Matplotlib

## 📁 Dataset
Student Performance Dataset

## 📈 Output
The heatmap shows strong correlations between student scores, especially between reading and writing scores.

## ▶️ How to Run the Project

1. Download the dataset.
2. Place the dataset in the same folder as the Python file.
3. Run the script:

```python
